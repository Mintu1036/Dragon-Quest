# 🐉 Dragon Quest

A 2D action-platformer built with Unity where players explore dangerous levels, defeat enemies, avoid deadly traps, and complete their quest through a fantasy world.

---

## 🎮 Gameplay

The player controls a hero capable of:

- Running and jumping
- Double jumping
- Wall jumping
- Casting fireballs
- Defeating enemies
- Avoiding environmental hazards
- Progressing through multiple levels

The game features combat, health management, level progression, menus, sound effects, and a game completion system.

---

## ✨ Features

### Player Mechanics
- Smooth character movement
- Jump system
- Double jump
- Wall jump
- Coyote Time
- Fireball projectile attacks

### Combat System
- Player health system
- Enemy health system
- Projectile combat

- Patrol enemies
- Spike Head enemy

- Main Menu
- Pause Menu
- Game Over Screen
- Game Completed Screen

### Audio
- Background music
- Sound effects
- Menu interaction sounds

## 🕹 Controls

| Action | Key |
|----------|----------|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Jump | Space |
| Attack | Left Mouse Button |
| Pause | Esc |
| Menu Navigation | W / S or Arrow Keys |
| Select Option | Enter |
